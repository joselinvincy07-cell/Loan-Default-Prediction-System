# Loan Default Prediction System

## Introduction

Loan default prediction is an important task in the banking and financial sector. Financial institutions face risks when borrowers fail to repay loans, resulting in financial losses and poor credit risk management.
This project uses the Lending Club Loan Dataset from Kaggle to analyze borrower behavior, identify risk factors, and predict loan-related patterns using Exploratory Data Analysis (EDA) and Linear Regression.

## Dataset Overview

**Dataset Name:** Lending Club Loan Data

**Source:** Kaggle

## Features Used:

* Loan Amount
* Loan Term
* Interest Rate
* Annual Income
* Employment Length
* Home Ownership
* Loan Purpose
* Loan Grade
* Loan Status

## Methodology

1. Data Collection
2. Data Cleaning
3.  Exploratory Data Analysis (EDA)
4.  Machine Learning

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Conclusion

This project analyzed the Lending Club Loan Dataset using EDA and Linear Regression to understand borrower behavior and loan default patterns.
The results showed that factors such as income, interest rate, and loan term influence loan repayment.
Overall, the project demonstrates how data analysis can support better credit risk assessment and loan approval decisions.
